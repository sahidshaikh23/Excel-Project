


# Excel Dashboard

## Overview
An Excel dashboard for visualizing and analyzing data through charts, graphs, and KPIs.

## Features
- **Dynamic charts** (bar, pie, line)
- **Auto-updating** with new data
- **Data filtering** with slicers
